# Analog-Active-Noise-Cancellation-System

An analog noise reduction system that minimizes unwanted ambient noise using **phase inversion** and **destructive interference**. The system captures external noise, generates an inverted (180° out-of-phase) signal through analog filtering and amplification stages, and combines it with the original noise to achieve real-time cancellation without digital signal processing.

This project demonstrates the practical implementation of **analog signal conditioning**, **active filtering**, and **operational amplifier circuits**, making it suitable for low-latency and low-power audio applications. The complete circuit was designed and validated using **Proteus**, followed by PCB layout development in **Altium Designer**.

## ✨ Features

- 🎤 Analog ambient noise acquisition
- 🔄 Phase inversion for anti-noise generation
- 🎚 Active filtering and signal conditioning
- 🔊 Real-time noise cancellation using destructive interference
- ⚡ Low-latency analog processing
- 🖥 Circuit simulation in Proteus
- 📐 PCB design using Altium Designer
