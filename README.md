# Analog-Active-Noise-Cancellation-System
Project Overview

The Analog Active Noise Cancellation (ANC) System is an analog signal processing project that reduces unwanted ambient noise by generating an anti-noise signal using the principle of phase inversion and destructive interference. Unlike digital ANC systems, this design performs real-time noise cancellation entirely through analog circuitry, resulting in low latency and minimal processing overhead.

The system captures environmental noise using a microphone, processes the signal through filtering and amplification stages, generates an inverted version of the noise waveform, and combines it with the original sound. When the two signals are superimposed, destructive interference significantly attenuates the unwanted noise while preserving the desired audio signal.

The circuit was designed, simulated, and verified using Proteus and translated into a PCB layout using Altium Designer, demonstrating the practical implementation of analog signal conditioning and noise cancellation techniques.
